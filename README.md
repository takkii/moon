ねこだるま（大きなねこ）が英語ならば、月は日本語で書きます。

Python3により簡潔に、リスト型を返したテキスト処理を読み込ませました。

従来通りDeinから入れる方法と、PyPIを利用する方法があります。

リスト型に内蔵で単語を生成しているので、高速も安定もあります。
